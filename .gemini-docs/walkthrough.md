# 遊戲資產工作流：從概念到互動實裝

本專案完美展示了利用 AI (Imagen 3) 定型風格，並整合至具備專業互動性的 Web 展示平台之完整管線。

## 核心成果展示

### 1. 風格定型 (Style Anchoring)
我們建立了 17 張高品質的吉卜力風格背景基準集。這些圖片具備統一的色彩與溫暖筆觸，作為後續資產生成的視覺基準。

### 2. 精緻資產生成 (Asset Generation)
利用「風格參考」技術生成了：
- **礦坑入口**：場景的主要環境資產。
- **神秘礦石**：具備 8 個平坦面的幾何體，適合進行多面貼圖展示。
- **吉卜力風寶箱**：點綴場景的互動產物。
- **古代符文圖集**：用於疊加在礦石上的點綴字體。
- **裂縫貼圖**：表現打擊回饋的圖層。

### 3. 互動展示平台 (Web Showcase)
一個基於 Vite 的高性能前端介面，具備以下功能：
- **視差滾動 (Parallax Scrolling)**：滑鼠感應的動態深度效果。
- **敲擊回饋 (Mining Mechanics)**：點擊礦石會觸發震動縮放與裂縫疊加。

### 4. 進階互動工具：3D 視覺化調整器 (Visual 3D Gizmo)
這是本專案的技術亮點，解決了 2D 貼圖在立體表面對齊的難度：
- **直覺拖拽**：在礦石旁直接透過 3D 平面預覽進行旋轉校正。
- **單一符文選取**：透過 UV 偏移與縮放，從符文集中精確選取單一符文。
- **全方位控制**：支援 X/Y/Z 三軸旋轉（Shift + 拖拽）及鍵盤方向鍵微調。
- **自動化 CSS**：調整完畢後自動輸出對應的 CSS 代碼以便保存。

## 如何開始與運行

1. **安裝環境**：
   ```bash
   npm install
   ```
2. **啟動開發伺服器**：
   ```bash
   npm run dev
   ```
3. **體驗地址**：[http://localhost:5173/](http://localhost:5173/)
   - 在頁面上直接點擊礦石進行「挖掘」。
   - 使用右側面板與 3D 工具進行符文微調。

## 技術棧總結
- **AI 端**：Imagen 3 (Style Reference, Single Object Generation)。
- **腳本端**：Python (OpenCV 去背處理、Batch 生成)。
- **前端端**：Vanilla JS (3D Transform, Parallax Logic, CSS Animations)。
